# Skills Catalog

Reusable multi-step workflow automations for Claude Code. Each skill becomes a slash command (e.g., `/add-endpoint`).

## Available Skills

> Coming soon. Skills will be added here as the ecosystem grows.

## How to Use

1. Copy the skill directory into your project: `cp -r <skill>/ your-project/.claude/skills/`
2. Run the skill in Claude Code: `/<skill-name>`

## How to Create Your Own

See the [skill template](../../advanced/.claude/skills/scaffold-feature/SKILL.md) for the standard 4-step structure (Gather → Validate → Execute → Verify).
