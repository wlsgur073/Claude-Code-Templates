# Agents Catalog

Specialized agent role definitions for Claude Code. Agents are scoped to specific parts of your codebase with restricted tool access.

## Available Agents

> Coming soon. Agent definitions will be added here as the ecosystem grows.

## How to Use

1. Copy the agent file into your project: `cp <agent>.md your-project/.claude/agents/`
2. Claude will automatically discover agents in `.claude/agents/`
