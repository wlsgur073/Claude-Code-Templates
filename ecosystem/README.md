# Ecosystem

Ready-to-use Claude Code components. Copy any component directly into your project's `.claude/` directory.

## Categories

| Directory | What's inside |
|-----------|---------------|
| [skills/](skills/) | Reusable multi-step workflow automations |
| [hooks/](hooks/) | Pre/post tool-use hook configurations |
| [agents/](agents/) | Specialized agent role definitions |

## Usage

```bash
# Example: copy a skill into your project
cp -r ecosystem/skills/<skill-name> your-project/.claude/skills/

# Example: merge a hook config into your settings.json
# (manually copy the hook entries into your .claude/settings.json)
```
