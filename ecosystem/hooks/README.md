# Hooks Catalog

Pre-built hook configurations for Claude Code. Hooks run shell commands before or after Claude uses a tool.

## Available Hooks

> Coming soon. Hook configurations will be added here as the ecosystem grows.

## How to Use

Copy the hook entries from a configuration file into your `.claude/settings.json` under the `hooks` key.

## How to Create Your Own

See the [advanced settings template](../../advanced/.claude/settings.json) for the hook structure, or the [Advanced Features Guide](../../guide/advanced-features-guide.md) for full documentation.
