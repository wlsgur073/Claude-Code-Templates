# 에이전트 카탈로그

Claude Code용 전문화된 에이전트 역할 정의. 에이전트는 코드베이스의 특정 부분에 범위가 제한되며 도구 접근이 제어됩니다.

## 사용 가능한 에이전트

> 곧 추가 예정. 생태계가 성장하면서 에이전트 정의가 추가됩니다.

## 사용법

1. 에이전트 파일을 프로젝트에 복사: `cp <agent>.md your-project/.claude/agents/`
2. Claude가 `.claude/agents/`의 에이전트를 자동으로 인식합니다

## 직접 만들기

표준 프론트매터 형식은 [에이전트 템플릿](../../advanced/.claude/agents/developer.md)을, 전체 문서는 [고급 기능 가이드](../../guide/advanced-features-guide.md)를 참고하세요.
