# 스킬 카탈로그

Claude Code용 재사용 가능한 멀티스텝 워크플로우 자동화. 각 스킬은 슬래시 명령어(예: `/add-endpoint`)로 실행됩니다.

## 사용 가능한 스킬

> 곧 추가 예정. 생태계가 성장하면서 스킬이 추가됩니다.

## 사용법

1. 스킬 디렉토리를 프로젝트에 복사: `cp -r <skill>/ your-project/.claude/skills/`
2. Claude Code에서 스킬 실행: `/<skill-name>`

## 직접 만들기

표준 4단계 구조(수집 → 검증 → 실행 → 확인)는 [스킬 템플릿](../../advanced/.claude/skills/scaffold-feature/SKILL.md)을 참고하세요.
