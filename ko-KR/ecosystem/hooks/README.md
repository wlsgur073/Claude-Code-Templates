# 훅 카탈로그

Claude Code용 사전 구성된 훅 설정. 훅은 Claude가 도구를 사용하기 전후에 셸 명령어를 실행합니다.

## 사용 가능한 훅

> 곧 추가 예정. 생태계가 성장하면서 훅 설정이 추가됩니다.

## 사용법

설정 파일의 훅 항목을 `.claude/settings.json`의 `hooks` 키 아래에 복사하세요.

## 직접 만들기

훅 구조는 [고급 설정 템플릿](../../advanced/.claude/settings.json)을, 전체 문서는 [고급 기능 가이드](../../guide/advanced-features-guide.md)를 참고하세요.
